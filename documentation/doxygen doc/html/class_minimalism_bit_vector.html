unsigned int  pWeight

Constructor de los objetos de la clase, inicializa las variables principales tales variables son aquellas que describen las principales caracteristicas de los objetos en memoria, tales caracteristicas son: el offset o direccion relativa, esta se mide en bytes y el corriemiento de bytes se da de 1 en 1, el peso del objeto y finalmente el tipo del objeto.

Parameters
pOffsetes la ubicacion del objeto que ademas es el identificador unico de cada una de las variables en memoria virtual, debido a que los objetos no pueden compartir la misma direccion de memoria.
pWeightes el tamanio del objeto al cual representa el objeto instanciado
pTypees el tipo del objeto al cual esta clase representa.
